# DVR Auto-Downloader

A Python script that automates the download of recorded media from a DVR recording service. It runs on a daily schedule, loads a saved browser session, navigates to the recordings list, and downloads any new titles — skipping anything already retrieved.

---

## How It Works

The script uses [Playwright](https://playwright.dev/python/) to drive a real browser session. Rather than logging in on every run, it loads a persistent browser session saved during initial setup. This allows it to bypass bot detection and navigate directly to the recordings list. It then checks each available title against a local download log and downloads anything new.

---

## Requirements

### Python Dependencies

Install dependencies using the project virtual environment:

```bash
pip install playwright python-dotenv
playwright install chromium
```

### Required Files

The following files must be present in the same directory as `downloader.py` before running the script:

| File/Folder | Purpose |
|---|---|
| `.env` | Stores configuration values (see below) |
| `downloaded.log` | Tracks previously downloaded titles to prevent re-downloading. Create as an empty file on first run. |
| `error.log` | Captures session expiry or page load failures. Create as an empty file on first run. |
| `session/` | Saved browser session generated by `setup_session.py`. Required for authentication. |

> ⚠️ `.env`, `downloaded.log`, `error.log`, and `session/` are excluded from version control via `.gitignore` and must be created manually on each machine.

---

## Configuration

Create a `.env` file in the project root with the following variables:

```
DVR_URL=https://www.fakedvrrecordingsurl.com/list/login
DVR_DOWNLOAD_DIR=/path/to/your/download/folder
```

---

## First-Time Setup

Before running the downloader for the first time, you must save a browser session by running:

```bash
python setup_session.py
```

A browser window will open. Log in manually, then return to the terminal and press Enter. The session will be saved to the `session/` folder and reused on every subsequent run.

### Session Expiry

If the session expires, the script will write a timestamped entry to `error.log` and exit. To resolve, simply run `setup_session.py` again to save a fresh session.

---

## Scheduled Execution

This script is designed to run once daily via cron. To configure, run `crontab -e` and add the following line, replacing the paths with your own:

```
0 9 * * * /home/your-username/PycharmProjects/dvr-script/.venv/bin/python /home/your-username/PycharmProjects/dvr-script/downloader.py
```

This runs the script every day at 9:00am. The full path to the virtual environment's Python binary is required so cron uses the correct interpreter with all dependencies available.

---

## Logging

- **`downloaded.log`** — appended with the recording ID of each successfully downloaded title
- **`error.log`** — appended with a timestamped entry if the session has expired or the recordings page fails to load
